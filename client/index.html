API Basejump: Timestamp Microservice

User Stories:
  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: Nvember 18, 2016)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://saintpeter-ts-micro.herokuapp.com/November%2018,%202016
https://saintpeter-ts-micro.herokuapp.com/20131040

Example output:

{"unix":"1479402000","natural":"November 18, 2016"}
{"unix": null, "natural": null }
